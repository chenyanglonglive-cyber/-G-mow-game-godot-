# MOW Tower Defense Godot

This is a Godot 4 migration prototype generated from the Phaser version.

Run it with:

```powershell
G:\GoDot4\Godot_v4.6.3-stable_win64.exe --path <project-path>
```

The `assets` folder is a Windows junction to:

```text
G:\OneDrive\mow-game-assets
```

Current migrated features:

- 450x800 portrait project
- Background art
- Two enemy paths
- Build slots
- Four tower types
- Basic tower upgrades
- Enemy waves
- Gold, shield and wave UI
- Bullets, frost slow and flame burn

This is a playable Godot base, not a pixel-perfect port of every Phaser visual effect yet.
