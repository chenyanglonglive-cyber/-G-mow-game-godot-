# Agent Rules: Plan-With-File Standard

Always follow the `plan-with-file` skill workflow for any task that involves modifying code files or creating new components.

- **Pre-execution Planning**: Before modifying any source files, research the codebase and write/update a `PLAN.md` file in the project root containing file touchpoints and step-by-step tasks.
- **Approval Checkpoint**: Explicitly present the plan to the user and wait for approval before editing any files.
- **Iterative Updates**: Update the task checklist in the plan file as tasks are implemented.

# Agent Rules: Chinese Language Standard for Artifacts

- **Language Requirement**: All generated planner artifacts, including `implementation_plan.md` (Implementation Plan), `task.md` (Task lists), and `walkthrough.md` (Walkthrough summaries), MUST be written in Chinese. Do not write them in English.
